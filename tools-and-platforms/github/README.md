# Github

## 基本git指令

```text
git status
git log
# basic
git add .
git commit -m "updata information"
git remote -v
>>>origin	git@github.com:s/exercise.git (fetch)
>>>origin	git@github.com:s/exercise.git (push)
git push origin master
git checkout .    恢复到未submit之前的状态，删除所有修改的操作
```
